# AI Internship Scam Detection System

## Overview
This project is an AI-based web application designed to detect fake internship offers using Natural Language Processing (NLP), Machine Learning, and OCR.

## Problem Statement
Many students fall victim to internship scams involving registration fees, urgency tactics, and guaranteed placements. This system helps students identify suspicious internship messages.

## Features
- Text-based internship analysis
- Screenshot upload with OCR extraction
- Scam pattern detection
- Confidence score generation
- Risk level classification
- Explainable AI reasoning

## Tech Stack
- Python
- Streamlit
- NLP
- Machine Learning
- Tesseract OCR

## How to Run
pip install -r requirements.txt
streamlit run app.py
